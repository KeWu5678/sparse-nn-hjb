# baseline — vdp

Convex reference point for the VDP studies: the best achievable relative H1
error vs. network size (neuron/H1 frontier) with **ReLU + L1** against the
non-convex sweeps (`../log_penalty`, `../frac_exp_penalty`). Findings are in
`results.md` (generated by `analysis.py`, dataset passed as `argv[1]`,
default `vdp`). Pendulum counterpart: `../../02_pendulum/baseline`.
